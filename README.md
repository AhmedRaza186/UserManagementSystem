# User Management System - Frontend

A modern, responsive frontend for the User Management System, built with vanilla web technologies. It provides a seamless interface for user registration, authentication, and community management.

## ✨ Features
- **Modern Landing Page**: High-quality design with feature highlights.
- **Authentication Pages**: Sleek login and signup forms with real-time validation.
- **OTP Verification**: Interactive 6-digit code entry with a 1-minute countdown timer.
- **Dynamic Dashboard**: View all community members with search and filter options.
- **Profile Management**: Logged-in users can update their name, age, phone, and profile picture.
- **Cloudinary Integration**: Effortless image uploads for profile pictures.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **HTML5** & **Vanilla CSS**
- **JavaScript** (Vanilla)
- **FontAwesome** (Icons)
- **Flatpickr** (Date picker)
- **Cloudinary** (Image hosting)

## 🏗️ Usage
1. Clone the repository.
2. Open `index.html` in your browser (or use a live server).
3. Ensure the backend URL in `auth.js` and `dashboard.js` is correct.
